# PaddleSeg高性能全场景模型部署方案—FastDeploy

## FastDeploy介绍

[FastDeploy](https://github.com/PaddlePaddle/FastDeploy)是一款全场景、易用灵活、极致高效的AI推理部署工具，使用FastDeploy可以简单高效的在10+款硬件上对PaddleSeg Matting模型进行快速部署
